# 🔥 Firebug Project Declaration

## 1. 프로젝트 개요
- **프로젝트명:** Firebug  
- **한 줄 요약:** 👉 초기 산불 감지용 초소형 AI 벌레 로봇 저비용 소모형
- **핵심 아이디어:** 👉 IOT + AI + Cloud + 드론
## 2. 문제 정의
👉 산불은 초기 진압 하지 못할 경우, 대규모 피해로 확산될 위험이 크다.  
👉 현재의 감시체계는 인력 중심으로 운영되어, 지형적 한계로 인한 감시 사각지대가 필연적으로 발생한다. 

## 3. 목표 및 성공 기준
| 목표 | 측정 지표 | 성공 기준 |
|------|------------|------------|
| IoT 센서 네트워크 구축 | 측정 오차 | ±3% 이내 |
| 이상 탐지 AI 모델 | 정확도 | 90% 이상 |
| 클라우드 전송 | 지연 | 5초 이하 |
| 시제품 완성 | 시연 가능 | 실제 데이터 데모 성공 |

## 4. 기술 구성
- **하드웨어:** ESP32-S3, SHT41, LoRa 모듈  
- **소프트웨어:** Arduino, Python, TensorFlow Lite  
- **클라우드:** Firebase, Streamlit 대시보드

## 5. 로드맵
| 기간 | 핵심 목표 | 산출물 |
|------|-------------|---------|
| 1개월차 | 환경 세팅, 센서 테스트 | Blink, Hello Sensor |
| 2개월차 | LoRa 통신 + AI MVP | 로컬 이상탐지 시스템 |
| 3개월차 | 클라우드 연동 | 실시간 대시보드 |
| 4개월차 | 통합 및 문서화 | 완성 데모, 백서 v1 |

## 6. 리스크 및 대응
| 리스크 | 대응 |
|---------|-------|
| 전력 문제 | 슬립모드, 태양광 보조 |
| 데이터 손실 | 버퍼링 + 재전송 알고리즘 |
| 오탐지 | 데이터 증강, 임계값 조정 |

## 7. 개인 동기
👉 지난 2025년에 산불을 초기 진압하지 못하여 전국으로 확산되어 오랜기간 많은 이들이 고통받은 뉴스를 보며, "왜 조기진압을 하지 못했을까"라는 생각을 하며 위 아이디어를 떠올리게 됨.
